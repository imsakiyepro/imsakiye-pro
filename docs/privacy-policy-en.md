## 1. Introduction

## 2. Personal Data We Collect

### 2.1 Location Data
**Data Collected:** GPS coordinates (latitude/longitude)

**Purpose:** To calculate prayer times specific to your location, determine Qibla direction, and show nearby mosques.

**Legal Basis:** Legitimate interest (GDPR Art. 6(1)(f)) - core functionality of the application

**Retention:** Location data is processed **only on your device** and is **never sent to any server**. It is used solely for local calculations.

**Sharing:** Not shared with third parties.

### 2.2 Statistical Data (Anonymous)
**Data Collected:** Prayer completion counts (aggregated statistics)

**Purpose:** To display global user statistics (e.g., "X people prayed Fajr today")

**Legal Basis:** Consent (GDPR Art. 6(1)(a)) - by using the application

**Retention:** Statistics are stored anonymously in Firebase Firestore. No user identification is associated with these counts.

**Sharing:** Stored on Firebase (Google Cloud) servers. As this is anonymous data, it is not considered personal data.

### 2.3 Notification Token
**Data Collected:** Device notification token

**Purpose:** To send prayer time reminders and in-app announcements.

**Legal Basis:** Legitimate interest (GDPR Art. 6(1)(f)) - enhancing user experience

**Retention:** Token is stored by Firebase Cloud Messaging (FCM) system.

**Sharing:** Shared with Google Firebase service (technical requirement for notification delivery).

### 2.4 Local Data (Stored on Device)
The following data is stored **only on your device** and is never sent to servers:
- Your prayer completion history (daily tracking)
- Missed prayer (Qada) counts
- App settings (notification preferences, theme, etc.)
- Selected city

**Deletion:** All local data is automatically deleted when you uninstall the application.
